Build
-----

    git clone git@github.com:DavidTruby/SpaceWorlds.git
    cd SpaceWorlds
    git submodule init && git submodule update
    mkdir build
    cd !$
    cmake ..
    make

####Requirements 

C++14 compiler (MinGW on Windows)  
[CMake](http://www.cmake.org/)  
[libsfml2.1](http://sfml-dev.org)  
[libboost](http://boost.org)
